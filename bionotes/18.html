Levels of Classification

Carolus Linnaeus created an accurate way of classifying organisms in a hierarchy. This was also used for binomial nomenclature, which is how organisms scientific names are given. The name is in the format genus, species. All levels in his system of classification are:

  1. kingdom
  2. phyllum
  3. class
  4. order
  5. family
  6. genus
  7. species
